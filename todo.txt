Тестовое задание:
Создать небольшое CRUD приложение.
Задание:
На странице отображается таблица (данные можно придумать любые) +++
 и форма для добавления нового значения. +++
В каждой колонке таблицы есть кнопки: удалить, изменить, +++
1.При первичной загрузке страницы, получить данные из бэка 
(можно использовать любые бд или даже файл), отобразить в таблице +++

2. При заполнении формы и нажатии кнопки отправить:
       2.1 Отправить данные из формы на бэк +++
       2.2 Записать данные на бэке +++
       2.3 Отправить на фронт сообщение о записи +++
       2.4 При успешной записи, на фронте очистить форму, +++
       показать сообщение о том, что запись сохранена, +++
        перезагрузить таблицу <---

       2.5 При неуспешной записи, показать сообщение об ошибке 

 3. При нажатии на кнопку удалить, котороя есть у каждой записи в таблице:
       3.1 Отправить запрос на бэк для удаления записи
       3.2 При успешном удалении, показать сообщение об этом, перезагрузить таблицу
       3.3 В случае ошибки, показать сообщение об ошибке 

4.   При нажатии на кнопку изменить, котороя есть у каждой записи в таблице:
       4.1 Заполнить форму данными из строки таблицы
       4.2 При нажатии на кнопку отправить, отправить данные из формы на бэк
       4.3 Изменить запись на бэке
       4.4 Отправить на фронт сообщение о изменении записи
       4.5 При успешной изменении записи , на фронте очистить форму, показать сообщение о том, что запись изменилась, перезагрузить таблицу
       4.6 При неуспешной изменении записи , показать сообщение об ошибке

Главное условия:
1) Бэк PHP,
2) Фронт JS. Внешний вид не важен, главное функциональность.
3) Проект разместить на GitHub с объяснением по запуску.
Дополнительно можно получить бонусы используя современный подход к фронтэнду
